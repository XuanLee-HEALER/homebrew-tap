# Homebrew Tap

Personal Homebrew tap by [@XuanLee-HEALER](https://github.com/XuanLee-HEALER).

## Usage

```bash
brew tap XuanLee-HEALER/tap
```

## Available Casks

| App | Install | Description |
|-----|---------|-------------|
| [blowup](https://github.com/XuanLee-HEALER/blowup) | `brew install --cask blowup` | Film discovery, download, subtitle management & knowledge base |
| [FeedMe](https://github.com/XuanLee-HEALER/FeedMe) | `brew install --cask feedme` | Lightweight macOS menu bar RSS reader |
